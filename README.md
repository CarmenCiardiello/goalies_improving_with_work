# Goalies Improving with Work

## Purpose

This project investigates the idea that hockey goalies need to face shots in order to perform well. The hypothesis is that goalies who have not had much work (i.e., faced few shots) up to a given point in a game are more liable to give up goals compared to goalies who have been kept active.

## Approach

The analysis will:

1. **Characterize the phenomenon** — Measure whether goalies who have faced fewer shots tend to allow goals above or below expected (goals saved above expected, or GsAX), using expected goals models to establish a baseline.
2. **Test consistency across seasons** — Examine whether this pattern holds for individual goalies across multiple seasons, determining if it is a persistent tendency or influenced by random variation.

## Inspiration

This idea has been discussed on [The Hockey PDOcast](https://podcasts.apple.com/us/podcast/the-hockey-pdocast/id1045850849) when Dimitri Filipovic brings on Kevin Woodley for their goalie deep dives.
